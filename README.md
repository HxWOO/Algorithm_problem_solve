### 에라토스테네스의 체
- 소수를 구하는 방법
- 숫자 n까지의 소수를 구하려면 소수 m (m <= n^1/2)의 배수를 모두 빼는식으로 체로 불순물을 거르듯이 걸러냄

### 유클리드 호제법
- 2개의 자연수의 최대공약수를 구하는 알고리즘
- 두 자연수 a, b 에 대해 a를 b로 나눈 나머지인 r이라 하면, a와 b의 최대공약수는 b와 r의 최대공약수와 같다.
- 따라서, 이 과정을 최대공약수가 나올때까지 재귀적으로 반복하면 최대공약수를 구할 수 있다.
- e.g. "클래스2 2609번"
